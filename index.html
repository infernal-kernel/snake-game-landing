Snake Game

Juego clásico tipo "Snake": controles personalizables, diseño limpio y partidas rápidas.

Descripción breve

Snake Game es un juego arcade sencillo pensado para partidas cortas. Cambia el tamaño y posición de los controles desde ajustes para adaptarlo a tu dedo.

Características

  • Controles movibles y ajustables
  • Guardar récords locales
  • Gráficos ligeros y jogabilidad fluida

Instrucciones rápidas para publicar el sitio

  1. Sube este archivo como index.html en tu hosting (GitHub Pages, Firebase Hosting, Netlify, Surge, etc.).
  2. Si ya tienes una política de privacidad pública, puedes usarla directamente. He actualizado el enlace a: https://infernal-kernel.github.io/snake-es-privacy/. Si no tienes una, crea también privacy.html (plantilla incluida más abajo) y súbela a la misma web.
  3. Asegúrate de que la URL sea HTTPS y accesible públicamente (Play Console lo requiere).
  4. En Play Console pega la URL del sitio (p. ej. https://tudominio.com) y, si corresponde, la URL de la política de privacidad (p. ej. https://tudominio.com/privacy.html).

Si no tienes dominio, usa GitHub Pages o Firebase Hosting (gratuitos) con HTTPS.

Privacidad y permisos

Si tu app no recopila datos fuera del dispositivo (solo almacenamiento local) normalmente no necesitas política de privacidad estricta, pero Google recomienda incluir una página de "Privacidad" explicando qué datos se usan. Resumen

Snake Game es un juego local que no recopila información personal del usuario fuera del dispositivo a menos que el usuario lo permita explícitamente (por ejemplo, mediante funciones adicionales como servicios en la nube o integraciones de terceros).

Qué datos recogemos

  • Datos locales: puntuaciones y preferencias guardadas en el dispositivo.
  • Datos opcionales: si el usuario elige integrar servicios externos (p. ej. Firebase), se informará y pedirá consentimiento específico.

Cómo usamos los datos

Los datos se usan únicamente para mejorar la experiencia de juego (guardar récords y configuraciones). No vendemos ni compartimos información personal con terceros para fines comerciales.
