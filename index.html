Descripción breve

Snake Game es un juego arcade sencillo pensado para partidas cortas. Cambia el tamaño y posición de los controles desde ajustes para adaptarlo a tu dedo.

Características

  • Controles movibles y ajustables
  • Guardar récords locales
  • Gráficos ligeros y jugabilidad fluida

Resumen

Snake Game es un juego local que no recopila información personal del usuario fuera del dispositivo a menos que el usuario lo permita explícitamente (por ejemplo, mediante funciones adicionales como servicios en la nube o integraciones de terceros).

Qué datos recogemos

  • Datos locales: puntuaciones y preferencias guardadas en el dispositivo.
  • Datos opcionales: si el usuario elige integrar servicios externos (p. ej. Firebase), se informará y pedirá consentimiento específico.

Cómo usamos los datos

Los datos se usan únicamente para mejorar la experiencia de juego (guardar récords y configuraciones). No vendemos ni compartimos información personal con terceros para fines comerciales.
